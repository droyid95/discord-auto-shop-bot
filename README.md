# 🛒 Discord Auto Shop Bot

Discord-бот магазина на Python + `disnake` с авто-выдачей товаров, балансом, категориями, продавцами, личным кабинетом и оплатой через YooMoney/CryptoPay.

## Главное

- `/start` открывает меню бота.
- `Магазин` показывает категории, подкатегории и товары.
- `Личный кабинет` показывает баланс, покупки и пополнение.
- `Админ-меню` доступно только администраторам из конфига.
- `Меню продавца` доступно продавцам и администраторам.
- Товары могут быть сообщением или файлом.
- Файлы можно добавлять через Discord-вложение или HTTPS-ссылку.
- Покупатель получает товар в ЛС.
- Если товар не удалось выдать в ЛС, баланс возвращается автоматически.

## Установка

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -U pip
python -m pip install -r requirements.txt
Copy-Item config.example.toml config.toml
```

После этого заполните `config.toml` и запустите бота:

```powershell
python main.py
```

## Основной конфиг

```toml
bot_token = "DISCORD_BOT_TOKEN"
admin_ids = [123456789012345678]
guild_id = 0
log_channel_id = 0

currency_name = "Рубль"
database_path = "data/shop.sqlite3"
products_path = "products"
max_product_file_mb = 350

bot_status = 1
activity_type = 1
activity_text = "магазин"
activity_stream_url = "https://www.twitch.tv/discord"

test_mode = false
accept_admin = false

yoomoney_enabled = false
yoomoney_token = "CHANGE_ME"
yoomoney_receiver = "410011234567890"
yoomoney_redirect_url = "https://discord.com"
yoomoney_payment_type = "SB"
yoomoney_poll_interval_seconds = 10
yoomoney_min_deposit = 5
yoomoney_max_deposit = 15000

cryptopay_enabled = false
cryptopay_token = "CHANGE_ME"
cryptopay_testnet = false
cryptopay_poll_interval_seconds = 10

seller_payout_enabled = false
seller_payout_percent = 65
seller_withdraw_delay_hours = 6
```

`guild_id = 0` включает глобальные команды и позволяет использовать `/start` в ЛС боту.

`test_mode = true` включает тестовый режим: пополнение зачисляется сразу, без реальной оплаты.

`accept_admin = true` включает модерацию действий продавцов: изменения отправляются администратору на принятие или отклонение.

## Способы оплаты

### YooMoney

Для YooMoney включите:

```toml
yoomoney_enabled = true
yoomoney_token = "TOKEN"
yoomoney_receiver = "4100..."
```

Токен получается через приложение YooMoney. В redirect URL можно указать `https://discord.com`.

### CryptoPay

Для CryptoPay включите:

```toml
cryptopay_enabled = true
cryptopay_token = "TOKEN"
cryptopay_testnet = false
```

Если токен создан в обычном `@CryptoBot`, оставьте:

```toml
cryptopay_testnet = false
```

Если токен создан в `@CryptoTestnetBot`, поставьте:

```toml
cryptopay_testnet = true
```

Токен от BotFather не подходит. Нужен именно токен Crypto Pay App.

## Промокоды

Промокоды создает только администратор:

`Админ-меню` -> `Создать промокод`

В форме указывается:

- имя промокода;
- процент бонуса;
- максимальное количество пользователей.

Промокод дает бонус к пополнению. Например, пользователь оплачивает `100`, промокод `+10%`, на баланс зачисляется `110`.

Когда лимит использований закончился, промокод становится неактивным. При вводе такого промокода бот покажет ошибку:

```text
Промокод больше неактивен.
```

## Личный кабинет

В `Личном кабинете` пользователь может:

- посмотреть баланс;
- пополнить баланс через YooMoney или CryptoPay;
- указать промокод при пополнении;
- посмотреть количество купленных товаров;
- посмотреть дату регистрации;
- посмотреть дату первой покупки;
- открыть список купленных товаров;
- повторно получить купленный товар и чек.

Если продавцам включены выплаты, продавец может подать заявку на вывод средств из ЛК.

## Категории и товары

Товары лежат в категориях и подкатегориях.

В магазине пользователь выбирает:

1. Категорию.
2. Подкатегорию.
3. Товар.
4. Количество, если товар ограниченный.
5. Подтверждение покупки.

В dropdown магазина показывается количество подкатегорий и товаров.

## Администратор

Администратор может:

- назначать и удалять продавцов;
- создавать и удалять категории;
- создавать и удалять подкатегории;
- выдавать, снимать и менять баланс;
- создавать промокоды;
- смотреть список ID товаров;
- удалять товары продавцов по ID;
- принимать или отклонять действия продавцов, если включен `accept_admin`.

Администраторы указываются в `config.toml`:

```toml
admin_ids = [123456789012345678]
```

## Продавец

Продавец может:

- создавать свои категории и подкатегории;
- создавать товары;
- редактировать только свои товары;
- смотреть ID своих товаров;
- добавлять payload к товарам;
- добавлять файлы через Discord-вложение или HTTPS-ссылку;
- получать процент с продаж, если это включено в конфиге;
- подавать заявку на вывод средств, если выплаты включены.

Типы товаров:

- `Сообщение` - покупателю отправляется текст.
- `Файл` - покупателю отправляется файл.
- `Вечный` - один payload можно выдавать много раз.
- `Ограниченный` - каждый payload выдается один раз.
